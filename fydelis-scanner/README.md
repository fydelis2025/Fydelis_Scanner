# Fydelis Security Scanner

O **Fydelis Security Scanner** é uma ferramenta de varredura de portas (port scanner) desenvolvida em Ruby, focada em performance e simplicidade. Ideal para reconhecimento básico em auditorias de segurança e testes de penetração.

## Funcionalidades
- Varredura assíncrona com suporte a múltiplas threads.
- Suporte a intervalos de portas (ex: 1-1024) e portas específicas.
- Banner Grabbing básico para identificação de serviços.
- Interface CLI intuitiva.

## Instalação

Certifique-se de ter o Ruby instalado. Clone o repositório e torne o arquivo executável:

```bash
git clone [https://github.com/SEU_USUARIO/fydelis-scanner.git](https://github.com/SEU_USUARIO/fydelis-scanner.git)
cd fydelis-scanner
chmod +x bin/fydelis
